В пакете Arrays:
    ArraysMethod - содержит операции с массивами.
    Задание 2.4 с интерфейсом IArraysOperation для перебора массивов.
    Остальные классы перебирают массив при помощи for, foreach, while, do...while.
В пакете loops:
    1. HomeWork1 - перемножение чисел.
    2. HomeWork2 - перемножение чисел рекурсией.
    3. HomeWork3 - перемножение цифр в числе.
    4. HomeWork4 - возведение в степень.
    5. HomeWork5 - вывод таблицы умножения.
В пакете presentation:
    1. HomeWork1 - нахождение наибольшей цифры числа.
    2. HomeWork2 - вероятность выпадения четного числа.
    3. HomeWork3 - количество четных и нечетных цифр числа.
    4. HomeWork4 - вывод ряда Фибоначчи.
    5. HomeWork5 - вывод последовательности чисел от минимального к максимальному с указанным шагом.
    6. HomeWork6 - вывод инвертированного числа.
    7. HomeWork7 - сумма четных и положительных элементов массива.
    8. HomeWork8 - вывод наибольшего элемента массива с четным индексом.
    9. HomeWork9 - вывод элементов, которые меньше среднего арифметическогоь.
    10. HomeWork10 - вывод двух наименьших элементов массива.
    11. HomeWork11 - удалить элементы массива принадлежащие интервалу.
    12. HomeWork12 - вывод суммы цифр элементов массива.
В пакете sortUtils:
    1. sortUtils - методы сортировки.
    2. sortUtilsTest - проверка методов с заранее и рандомно заполненными массивами.
В пакете utils:
    ArrayUtils - методы заполнения массива рандомно и с консоли.
